Local-only proxy helper

Claude iOS Session Cleaner

解决 Claude iOS 出现 “Something went wrong” 的登录异常问题。

1

启动服务

点击启动代理，软件会在这台电脑上开启本地代理服务。

2

iPhone 设置

请按下面每一步完成手机端设置。

A
连接同一个 Wi-Fi

让 iPhone 和这台电脑连接同一个 Wi-Fi。不要使用 iPhone 蜂窝网络，也不要用这台 iPhone 给电脑开热点。

B
设置 Wi-Fi HTTP Proxy

打开 iPhone：设置 -> Wi-Fi -> 当前网络 -> HTTP 代理 -> 手动。

填写 Server: 198.18.0.1　Port: 9091，然后点保存。

C
打开证书页面

在 iPhone Safari 打开 http://mitm.it。

D
安装并信任证书

安装 iOS 证书后，进入：设置 -> 通用 -> 关于本机 -> 证书信任设置，开启完全信任。

E
重开 Claude iOS

强退 Claude iOS，再重新打开。完成后看第 3 步进度。
